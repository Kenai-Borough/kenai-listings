# Kenai Listings

Kenai Listings is a fast Vite + React + TypeScript classifieds marketplace for the Kenai Peninsula, covering gear, jobs, services, housing, community posts, barter, and free stuff.

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- React Router, Framer Motion, Recharts, Leaflet
- Supabase client wiring and schema

## Local development
```bash
npm install
npm run dev
npm run build
```

## Deployment
GitHub Pages deploys automatically from `.github/workflows/deploy.yml` and serves `kenailistings.com` via `public/CNAME`.
